1. npm 회원 가입

2. npm login (on terminal)
   npm whoami (로그인 상태 확인)

3. 배포할 패키지를 git repository에 업로드
   git init
   git remote add origin [repository address]

4. package.json
   npm init --yes   (npm init)

    name : 필수
    version (1.0.0) : 필수
    description (README.md)
    main (index.js)
    scripts (test)
    keywords
    author
    license (ISC)
    bugs
    homepage

5. 버전  관리
   a.b.c
   a : major update
   b : minor update
   c : bug fixed update

6. 패키지 테스트 및 배포
   npm install 
   npm publish --access public
